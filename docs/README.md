# HadronRNG-Pass

Secure random generator and password generation library written in Rust.

HadronRNG-Pass provides a modular system for:

- Secure random byte generation
- Password generation
- Cryptographic key generation
- Entropy management
- Validation and security policies


## Features

- Fast random generation
- Configurable password policies
- Custom character sets
- Key generation
- Entropy collection
- Memory protection layer
- Multi-thread testing
- Stress testing
- Statistical randomness testing
