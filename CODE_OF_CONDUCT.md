Code of Conduct Our Pledge 

We are committed to making participation in the HadronRNG project welcoming, respectful and constructive for everyone.

We expect all contributors, maintainers and community members to act professionally and respectfully.

Expected Behavior 

Please:

Be respectful. Be constructive. Be patient. Be honest. Focus on technical discussions. Help others learn. Accept constructive feedback. Unacceptable Behavior 

The following behaviors are not acceptable:

Harassment or discrimination. Personal attacks or insults. Hate speech. Threats or intimidation. Spam or advertising. Deliberate disruption of discussions. Publishing private information without permission. Trolling or provoking conflicts. Communication 

When discussing code or design:

Critique ideas, not people. Explain technical decisions clearly. Respect different opinions. Keep discussions focused on the project. Reporting Issues 

If you experience or witness unacceptable behavior, please contact the project maintainer privately.

Reports will be handled confidentially whenever possible.

Enforcement 

Project maintainers may:

Request changes in behavior. Remove inappropriate content. Close discussions that become unproductive. Temporarily or permanently restrict participation for repeated or severe violations. 

Enforcement decisions are made at the maintainers' discretion.

Scope 

This Code of Conduct applies to:

GitHub Issues Pull Requests Discussions Project documentation Any official communication channels related to HadronRNG Attribution 

This document is inspired by widely used open-source community guidelines and adapted for the HadronRNG project.

Thank you for helping keep the HadronRNG community respectful, collaborative and focused on building secure software.
