If you believe you have discovered a security vulnerability in HadronRNG, please report it responsibly.

Please do not create a public GitHub issue for security vulnerabilities.

Instead:

Contact the project maintainer privately. Include a detailed description of the issue. Provide steps to reproduce the problem. Include affected version(s), operating system and Rust version. If possible, provide a proof of concept. 

The maintainer will acknowledge the report as soon as possible and begin an investigation.

Response Process 

Every security report is handled according to the following process:

Acknowledge the report. Verify the vulnerability. Assess its severity. Develop and test a fix. Publish a patched release. Publicly disclose the issue after a fix is available. Supported Platforms
